Stephen está tratando de leer un trozo de texto complicado y retorcido. Al ver esta situación, Nikola y Sophia decidieron ayudar de manera secreta. Para esto, crearon una regla de codificación simple para todos los mensajes expedidos, que deberá permanecer oculta para Stephan. ¡Vamos a ayudarles a completar su algoritmo!

Se te dará un texto, cuyas palabras contienen sólo letras, números o dígitos. De esta forma, "2045th" corresponde a dos elementos- el número 2045 y la palabra "th". Las palabras / números pueden estar separados por espacios en blanco y signos de puntuación !#$%&()*+,-./:;<=>?@[\]^_`{|}~. El texto tiene una sola línea (sin saltos de línea). Deberás codificar y mezclar el texto utilizando las siguientes reglas:

  • Invierte cada palabra
    "Hello" ⇒ "olleH".
  • Cambia mayúsculas a minúsculas y viceversa.
    "AsD" ⇒ "aSd".
  • Cambia cada dígito en un número, con la diferencia entre 9 y el dígito.
    "140" ⇒ "859" (9-1=8, 9-4=5, 9-0=9).
  • Reemplaza algunos signos de puntuación de acuerdo a la siguiente lista:
    • "," ⇔ "."
    • "!" ⇔ "?"
    • "(" ⇔ ")"
    • "{" ⇔ "}"
    • "[" ⇔ "]"
    • "<" ⇔ ">"
    • "@" ⇔ "#"
    • ";" ⇔ ":"
  • Trunca varios espacios en blanco consecutivos en uno.
    "      " ⇒ " ".

Sin embargo hay una restricción. Para evitar que Stephen se percate de la situación, tu código no deberá tener más de 12 lineas y no más de 80 símbolos en cada línea. "\ t" será contabilizado como 4 símbolos.

El objetivo principal Nikola y de Sophia es divertirse un poco, por lo que en esta misión deberás obtener el máximo número de errores PEP8. Verificaremos tu código con un módulo PEP8 (version 1.5.7) y por lo tanto, los requisitos PEP8 no son demasiado importantes aquí. Echa un vistazo al módulo, que es muy importante para esta misión.
Por cada error ("EXYZ") obtendrás 2 puntos y por cada advertencia ("WXYZ") 1 punto.

Datos de Entrada: Un texto, como una cadena (unicode).

Salida: El texto codificado, como una cadena (str).

Ejemplo:

twist("Hello World!") == "OLLEh DLROw?"
twist("I`m 1st") == "i`M 8TS"
twist("How are you? 905th.") == "WOh ERA UOY! 094HT,"
twist("The code - ([{<;#>}])") == "EHt EDOC - )]}>:@<{[("
twist("EMAIL        a@b.ru") == "liame A#B,UR"
twist(";-) 0_0 @__@") == ":-( 9_9 #__#"

¿Cómo se usa?: Echemos un vistazo más cuidadoso al PEP8 más cuidadosos y examinemos el módulo de verificación disponible. Si encuentras algunos errores (bugs), podrías ayudar al progreso de esta importante biblioteca para la comunidad Python. Puedes reportar problemas o enviar parches para la biblioteca PEP8 a través de Github.

For Stephan is hard to read an twisted text. So Nicola and Sophia created the simple encoding rules for messages which should be hidden from Stephan. Let's realize their algorithm.

You are given a text. Words contain only letters, numbers only digits. So "2045th" is two constructions - the number 2045 and the word "th". Words/numbers can be separated by whitespaces and punctuation symbols !#$%&()*+,-./:;<=>?@[\]^_`{|}~. The text has only one line without newlines. You should encode and twist the text with the next rules:

  • Reverse each word
    "Hello". ⇒ "olleH".
  • Swap letter case, uppercase to lowercase and versus.
    "AsD" ⇒ "aSd".
  • Change each digit in numbers to the difference between 9 and the digit.
    "140" ⇒ "859" (9-1=8, 9-4=5, 9-0=9).
  • Replace some punctuation symbols to each other with the follow list:
    • "," ⇔ "."
    • "!" ⇔ "?"
    • "(" ⇔ ")"
    • "{" ⇔ "}"
    • "[" ⇔ "]"
    • "<" ⇔ ">"
    • "@" ⇔ "#"
    • ";" ⇔ ":"
  • Truncate several consecutive whitespaces in one.
    "      " ⇒ " ".

But one restriction. Your code should have no more than 12 lines with no more than 80 symbols in each line. "\t" is counted as 4 symbols.

And the main goal and fun. In this mission yoo should get the maximum PEP8 errors. We will check your code with the pep8 module (version 1.5.7). So the PEP8 requirements are not too important here. Look at the module, it's more important for this mission.
For each error ("EXYZ") you will get 2 points, for each warning ("WXYZ") -- 1 points.

Input: A text as a string.

Output: The twisted text as a string.

Example: 

twist("Hello World!") == "OLLEh DLROw?"
twist("I`m 1st") == "i`M 8TS"
twist("How are you? 905th.") == "WOh ERA UOY! 094HT,"
twist("The code - ([{<;#>}])") == "EHt EDOC - )]}>:@<{[("
twist("EMAIL        a@b.ru") == "liame A#B,UR"
twist(";-) 0_0 @__@") == ":-( 9_9 #__#"
